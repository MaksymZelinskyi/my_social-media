# 🚀 My social Media

A social media api with a basic frontend built on Angular

## 🛠️ Features

- ✅ User authentication with Spring Security
- ✨ View feeds
- 📝 Create, edit, and delete publications
- ❤️ Like & comment publications


## 🧰 Tech Stack

- **Java 17**
- **Spring Boot**
- **PostgreSQL**
- **Hibernate**
- **Docker**
- **Angular & Typescript** for frontend
